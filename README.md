# oshikatsu_app
グッズの管理が苦手な人が、写真を撮るだけで簡単に整理できるようにするアプリ。（２次元限定）

# 環境構築

# 機能一覧
- AIでアイテム検索（写真から文字を読み取る→読み取った文字をもとに自動で検索)
- ログイン/ログアウト機能
- アイテムの登録（ユーザーベース）
    - 作品名
    - 名前（例：キャラ名）
    - 写真
    - カテゴリー
    - 個数
    - 写真から読み取った情報を追加して保存
- アイテム一覧
    - カテゴリーごと
    - 文字やタグで細かく検索
- 写真を使った検索
- アイテムの編集・削除
- アカウント情報の変更
